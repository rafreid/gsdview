# Plan 06-03: Manual refresh button

## Goal
Add a refresh button to the toolbar for manual graph reload.

## Tasks

1. Add refresh button to toolbar HTML
2. Style the button consistently
3. Wire up click handler to reload project
4. Show loading state during refresh

## Implementation

### HTML:
```html
<button id="refresh-btn" title="Refresh graph">↻ Refresh</button>
```

### JavaScript:
```javascript
document.getElementById('refresh-btn').addEventListener('click', async () => {
  if (selectedProjectPath) {
    const btn = document.getElementById('refresh-btn');
    btn.disabled = true;
    btn.textContent = 'Refreshing...';
    await loadProject(selectedProjectPath);
    btn.disabled = false;
    btn.textContent = '↻ Refresh';
  }
});
```

## Verification
- Load a project
- Click Refresh button
- Graph reloads with latest data
