# Plan 06-05: Window state persistence

## Goal
Remember window size and position between sessions.

## Tasks

1. Save window bounds on close
2. Restore window bounds on launch
3. Handle multi-monitor edge cases

## Implementation

### Main process:
```javascript
const Store = require('electron-store');
const store = new Store();

function createWindow() {
  const windowState = store.get('windowState', {
    width: 1200,
    height: 800
  });

  mainWindow = new BrowserWindow({
    width: windowState.width,
    height: windowState.height,
    x: windowState.x,
    y: windowState.y,
    webPreferences: {
      preload: path.join(__dirname, 'preload.js'),
      contextIsolation: true,
      nodeIntegration: false
    }
  });

  // Save window state on close
  mainWindow.on('close', () => {
    const bounds = mainWindow.getBounds();
    store.set('windowState', bounds);
  });

  // ...rest of createWindow
}
```

## Verification
- Resize window
- Close and reopen app
- Window restores to previous size/position
