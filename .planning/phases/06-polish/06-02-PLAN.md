# Plan 06-02: Auto-refresh on file changes

## Goal
When files in .planning/ change, automatically reload and re-render the graph.

## Tasks

1. Listen for files-changed events in renderer
2. Call loadProject to reload data
3. Show subtle notification of refresh
4. Preserve camera position during refresh if possible

## Implementation

### Renderer:
```javascript
// Listen for file changes
if (window.electronAPI && window.electronAPI.onFilesChanged) {
  window.electronAPI.onFilesChanged((event, data) => {
    console.log('Files changed:', data);
    if (selectedProjectPath) {
      showRefreshIndicator();
      loadProject(selectedProjectPath);
    }
  });
}

function showRefreshIndicator() {
  const indicator = document.getElementById('refresh-indicator');
  if (indicator) {
    indicator.classList.add('visible');
    setTimeout(() => indicator.classList.remove('visible'), 1500);
  }
}
```

### HTML:
```html
<div id="refresh-indicator">Refreshing...</div>
```

### CSS:
```css
#refresh-indicator {
  position: fixed;
  top: 60px;
  right: 20px;
  background: rgba(78, 205, 196, 0.9);
  color: #1a1a2e;
  padding: 8px 16px;
  border-radius: 4px;
  font-size: 12px;
  opacity: 0;
  transition: opacity 0.3s;
}

#refresh-indicator.visible {
  opacity: 1;
}
```
