# Plan 05-03: Hover tooltips

## Goal
Hovering over a node shows a tooltip with quick summary information.

## Tasks

1. Create tooltip HTML element
2. Add CSS for tooltip styling
3. Track mouse position on hover
4. Show tooltip near cursor with node info
5. Hide tooltip when mouse leaves node

## Implementation

The 3d-force-graph library has built-in nodeLabel which we're already using.
Enhance it with better formatting and add a custom tooltip for richer display.

### HTML:
```html
<div id="tooltip"></div>
```

### CSS:
```css
#tooltip {
  position: fixed;
  padding: 8px 12px;
  background: rgba(0, 0, 0, 0.85);
  color: #fff;
  border-radius: 4px;
  font-size: 12px;
  pointer-events: none;
  z-index: 1001;
  max-width: 250px;
  display: none;
  border: 1px solid rgba(255,255,255,0.2);
}

#tooltip.visible {
  display: block;
}
```

### JavaScript:
```javascript
const tooltip = document.getElementById('tooltip');

Graph.onNodeHover(node => {
  if (node) {
    let content = `<strong>${node.name}</strong><br>`;
    content += `<span style="color: ${getNodeColor(node)}">Type: ${node.type}</span>`;
    if (node.status) content += `<br>Status: ${node.status}`;
    tooltip.innerHTML = content;
    tooltip.classList.add('visible');
  } else {
    tooltip.classList.remove('visible');
  }
});

// Track mouse for tooltip positioning
container.addEventListener('mousemove', (e) => {
  tooltip.style.left = e.clientX + 15 + 'px';
  tooltip.style.top = e.clientY + 15 + 'px';
});
```

## Verification
- Hover over any node
- Tooltip appears near cursor with node info
- Tooltip disappears when mouse moves away
